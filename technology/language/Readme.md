# 简介

本文件夹存储各种编程语言（如Java、C、GO）、脚本语言（如shell、python）、标记语言（如mermaid、Latex）的相关文档，内容包括但不限于语言本身和该语言的框架。

## 文件夹说明

c++：C++ 和 C 的文档，由于 C++ 包括 C 的大部分特性，所以把他们归为一类。

comprehensive：语言之间的性能比较

goland：GO语言文档

java：Java语言文档

javascript：js语言文档

markup-language：编译语言文档，如mermaid、Latex等

python：python语言文档

shell：shell语言文档
